## Phase Creation Rules

1. **3-5 phases per milestone** - More phases = more overhead
2. **Each phase has clear deliverable** - What can you demo?
3. **Dependencies flow forward** - Phase N shouldn't depend on Phase N+1
4. **Vertical slices preferred** - Complete features, not horizontal layers
